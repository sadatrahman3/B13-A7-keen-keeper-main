# KeenKeeper

KeenKeeper is a responsive friendship tracking app built for the assignment brief. It helps users monitor how long it has been since they last contacted a friend, open a detailed relationship view, log quick check-ins, review a shared interaction timeline, and see basic friendship analytics.

## Technologies Used

- React
- React Router DOM
- Recharts
- Vite
- CSS

## Features

- Branded home page with summary cards, responsive friend grid, and loading state while local JSON data is fetched
- Friend details page with two-column layout, status styling, goal cards, and quick check-in actions that create timeline entries with toast feedback
- Timeline and analytics pages with filtering, sorting, persistent interaction history, and a pie chart breakdown of call, text, and video activity

## Pages

- Home
- Friend Details
- Timeline
- Stats
- 404 Not Found
